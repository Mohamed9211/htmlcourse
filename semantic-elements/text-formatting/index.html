Web Development Terminologies

Web Development Terminology

HTML
HyperText Markup Language – the standard language for creating web pages.
CSS
Cascading Style Sheets – used to style the appearance of HTML elements.
JavaScript
A scripting language used to create dynamic and interactive web content.
DOM
Document Object Model – a programming interface for HTML and XML documents.
Responsive Design
An approach to web design that ensures content looks good on all devices.
Media Query
A CSS technique used to apply styles depending on device characteristics like screen size.
Semantic HTML
Using HTML elements according to their meaning, like <header>, <article>, etc.
Flexbox
A CSS layout model for arranging elements in a flexible and responsive way.
Grid
A powerful CSS layout system for creating 2D layouts using rows and columns.
API
Application Programming Interface – allows communication between different software components.
AJAX
Asynchronous JavaScript and XML – allows web pages to update without reloading.
Bootstrap
A popular CSS framework for building responsive and mobile-first websites quickly.
Web Hosting
A service that provides storage space for websites on the internet.
Git
A version control system for tracking changes in source code.
GitHub
A platform for hosting and collaborating on Git repositories.
HTTP
HyperText Transfer Protocol – the protocol used for transferring web pages on the internet.
HTTPS
Secure version of HTTP, using encryption for secure communication over the web.
URL
Uniform Resource Locator – the address used to access a resource on the web.
SEO
Search Engine Optimization – techniques to improve website visibility in search engines.
Favicon
A small icon displayed in the browser tab, representing a website.
